# unifi_static_dns (Data Source)

Retrieves information about an existing UniFi static DNS record. Lookup by either id or key (hostname).

~> **Note:** This data source uses the v2 API and requires a UDM device.

## Example Usage

```terraform
# Look up a static DNS record by key (hostname)
data "unifi_static_dns" "example" {
  key = "server.local"
}

# Or look up by ID
data "unifi_static_dns" "by_id" {
  id = "60a1b2c3d4e5f67890123456"
}

# Use the data source
output "dns_value" {
  value = data.unifi_static_dns.example.value
}
```

## Schema

### Optional

- `id` (String) The unique identifier of the static DNS record. Specify either id or key.
- `key` (String) The hostname or domain name for the DNS record. Specify either id or key.

### Read-Only

- `enabled` (Boolean) Whether the DNS record is enabled.
- `port` (Number) Port number for SRV records.
- `priority` (Number) Priority value for MX and SRV records.
- `record_type` (String) The DNS record type (A, AAAA, CNAME, MX, NS, TXT, SRV).
- `ttl` (Number) Time to live in seconds for the DNS record.
- `value` (String) The value for the DNS record (IP address, hostname, or other value depending on record type).
- `weight` (Number) Weight value for SRV records.
