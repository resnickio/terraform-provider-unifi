# unifi_wlan (Data Source)

Retrieves information about an existing UniFi wireless network (SSID). Lookup by either id or name.

## Example Usage

### Lookup by Name

```hcl
data "unifi_wlan" "corporate" {
  name = "Corporate WiFi"
}

output "wlan_security" {
  value = data.unifi_wlan.corporate.security
}
```

### Lookup by ID

```hcl
data "unifi_wlan" "by_id" {
  id = "60a1b2c3d4e5f67890123456"
}
```
