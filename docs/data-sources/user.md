# unifi_user (Data Source)

Retrieves information about an existing UniFi user (client device record). Lookup by either id or mac.

## Example Usage

### Lookup by MAC Address

```hcl
data "unifi_user" "server" {
  mac = "aa:bb:cc:dd:ee:01"
}

output "server_ip" {
  value = data.unifi_user.server.fixed_ip
}
```

### Lookup by ID

```hcl
data "unifi_user" "by_id" {
  id = "60a1b2c3d4e5f67890123456"
}
```

## Schema

### Optional

- `id` (String) The unique identifier of the user. Specify either id or mac.
- `mac` (String) The MAC address of the client device (format: aa:bb:cc:dd:ee:ff). Specify either id or mac.

### Read-Only

- `blocked` (Boolean) Whether the device is blocked from network access.
- `first_seen` (Number) Unix timestamp when the device was first seen.
- `fixed_ip` (String) The fixed IP address for DHCP reservation.
- `hostname` (String) The hostname of the device from DHCP.
- `ip` (String) The current IP address of the device.
- `last_seen` (Number) Unix timestamp when the device was last seen.
- `local_dns_record` (String) A local DNS hostname record for this device.
- `local_dns_record_enabled` (Boolean) Whether the local DNS record is enabled.
- `name` (String) A friendly name for the client device.
- `network_id` (String) The network ID for the fixed IP DHCP reservation.
- `note` (String) Notes or description for the client device.
- `noted` (Boolean) Whether the device has a note.
- `oui` (String) The manufacturer OUI code derived from the MAC address.
- `site_id` (String) The site ID where the user exists.
- `use_fixed_ip` (Boolean) Whether DHCP reservation is enabled for this device.
- `usergroup_id` (String) The user group ID for bandwidth limiting.
