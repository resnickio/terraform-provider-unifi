# unifi_static_route (Data Source)

Retrieves information about an existing UniFi static route. Lookup by either id or name.

## Example Usage

### Lookup by Name

```hcl
data "unifi_static_route" "vpn_route" {
  name = "VPN Route"
}

output "vpn_network" {
  value = data.unifi_static_route.vpn_route.network
}
```

### Lookup by ID

```hcl
data "unifi_static_route" "by_id" {
  id = "60a1b2c3d4e5f67890123456"
}
```

## Schema

### Optional

- `id` (String) The unique identifier of the static route. Specify either id or name.
- `name` (String) The name of the static route. Specify either id or name.

### Read-Only

- `enabled` (Boolean) Whether the static route is enabled.
- `site_id` (String) The site ID where the static route exists.
- `static_route_distance` (Number) The administrative distance (metric) for the route.
- `static_route_interface` (String) The interface for the route (for interface routes).
- `static_route_network` (String) The destination network in CIDR notation.
- `static_route_nexthop` (String) The next hop IP address for the route.
- `static_route_type` (String) The static route type (nexthop-route, interface-route, blackhole).
- `type` (String) The type of route (static-route, interface-route).
